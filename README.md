<h1>Investigating the Impact of Online Learning Platforms on Student Performance</h1>

<h2>🎯 Project Overview</h2>
This project investigates how the Open University’s new Virtual Learning Environment (VLE) influences student academic performance. Using the Open University Learning Analytics Dataset, the analysis explores:

- <b>Whether VLE engagement improves grades</b>
- <b>How demographic factors relate to performance</b>
- <b>Weekly activity patterns</b>
- <b>Predictive modelling of student grades</b>
- <b>Statistical evidence of VLE effectiveness</b>

The project combines **data wrangling, exploratory data analysis, hypothesis testing,** and **linear regression** modelling to uncover insights and provide actionable recommendations.
<br />
<h2>Languages and Libraries</h2>

- Python (Pandas, NumPy, Matplotlib, Seaborn)

<h2>Environments Used </h2>

- Jupyter Notebook

<h2>📊 Datasets Used</h2>
The analysis uses seven datasets from the OU Learning Analytics collection, including:

- Courses
- Assessments
- Student Assessments
- Student Info
- Student Registration
- Student VLE Interactions
- VLE Activity Metadata

<h2>Key Findings</h2>
✔ <b>High VLE interaction improves grades</b>b>
 
 - Students with high interaction achieved 56.69% mean weighted grades vs 43.31% for low‑interaction students.

✔ <b>Module difficulty varies</b>
 - Some modules (e.g., AAA, DDD) show lower average scores and higher fail rates.

✔ <b>Engagement peaks early and mid‑course</b>
 - Weeks 2, 4, and 18 show the highest VLE activity.

✔ <b>Demographics show interesting patterns</b>
 - Female students have slightly higher fail rates
 - Ages 35–55 achieve the most distinctions
