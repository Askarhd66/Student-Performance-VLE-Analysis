<h1>Investigating the Impact of Online Learning Platforms on Student Performance</h1>

<h2>Description</h2>

This project utilizes statistical analysis on the Open University Learning Analytics Dataset (OULAD) to evaluate the impact of Virtual Learning Environments (VLE) on student academic outcomes. By analyzing a massive dataset of 177,410 records, the research employs rigorous hypothesis testing to determine if platform engagement directly correlates with higher grades, providing evidence-based insights to optimize digital learning strategies.
